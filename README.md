CS-370 Project Two, Reinforcement Learning Agent

    This project uses a Deep Q Network to train an intelligent agent to navigate a custom Pirate Treasure Maze. The notebook shows the full workflow, including the maze setup, the reward structure and the training loop. The HTML export is included so the project can be viewed without running the code. 
    For this project, I was given starter code that defined the maze, the game environment and the experience replay system. I created the rest of the code for the training loop, the testing process and the adjustments to the reward structure. I also made changes to print out the win rate, loss values and timing for each epoch. These updates helped me understand how the agent was improving across training process.
    Working on this assignment helped me understand the larger picture of what computer scientists do. Much of the work involves breaking down a complicated problem into smaller steps, testing different approaches and learning from mistakes. Computer scientists build tools and systems that solve real problems, and this matters because these tools shape the way people interact with technology. I learned how important it is to understand the environment, the state and the feedback the agent receives, because every part of that structure affects the final performance.
    This course also changed how i approach problems as a computer scientist. I learned not to jump straight to the solution, but to think more about how an agent learns, how it interacts with the world and how rewards guide its behavior. Instead of looking at a project only as code to finish, I focused on how the system makes decisions and how the model improves over time. That shift helped me slow down and see each part of the project with a clearer purpose.
    I also learned more about my ethical responsibilities in this field. Building intelligent systems means handling data carefully, respecting user privacy and making sure the model does not create harm through bias or poor design. The work in this course reminded me that systems need to be fair, secure and transparent so the people who use them can trust the results. As I continue learning in computer science, these responsibilities will guide how I design and evaluate future projects.
